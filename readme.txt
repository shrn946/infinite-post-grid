Description
The "Infinite Masonry Post Grid With Loading Effect" plugin enhances your WordPress website by creating an infinite scrollable post grid with a loading effect. This dynamic grid allows users to seamlessly browse through posts without needing to navigate through pages manually. It provides a visually appealing way to showcase your content and engage your audience.

Features
Infinite Scroll: Automatically loads more posts as the user scrolls down, providing a smooth browsing experience.
Masonry Layout: Displays posts in a responsive masonry grid layout, optimizing space and aesthetics.
Loading Effect: Enhances user experience with a loading effect, indicating the retrieval of additional posts.
Customization: Easily customize the display of the post grid using shortcode parameters.
Shortcode Integration: Simple shortcode implementation allows for flexible usage and placement within your website.
Shortcode Usage
To display the post grid, use the following shortcode:

[post_grid]

You can customize the display by adding optional parameters:

order: Specify the order of posts (e.g., "DESC" for descending order).
exclude_category_slug: Exclude posts from a specific category identified by its slug.
Example with parameters:

csharp
Copy code
[post_grid order="DESC" exclude_category_slug="category-slug"]
Feel free to adjust the shortcode attributes based on your preferences and requirements.

Installation
Upload the "Infinite Masonry Post Grid With Loading Effect" plugin folder to the /wp-content/plugins/ directory of your WordPress installation.
Activate the plugin through the 'Plugins' menu in WordPress.
Configure the plugin settings as desired.
Use the provided shortcode to display the post grid on your website.
